# PM2.5-Lung-Cancer-Data-Analysis
该项目源自本人2024-25年的SRT，由于懒得发论文，没什么创意且方向也不太匹配，故于此整理上传，主要是进行研究与论文工作的入门

Environmental dataset analysis with correlation analysis，regression and spatial methods (Moran’s I), showcasing data handling and quantitative skills that could transferable to astrophysical research.

使用相关分析，回归和空间方法（Moran’s I）进行环境数据集分析，展示数据处理和定量分析技能，这些技能可迁移到天体物理学研究。
